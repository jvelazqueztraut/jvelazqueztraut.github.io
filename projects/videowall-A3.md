Videowall installation and software development for [Area Tres Cowork](http://www.areatresworkplace.com/){:target="_blank" rel="noopener"}.

The hardware setup for this videowall allows all the TVs to be synchronized with the same content or all of them working individually. I developed a custom videowall player that handles many types of media using [openFrameworks](https://openframeworks.cc/){:target="_blank" rel="noopener"}: web rendering, offline-video, Youtube videos and playlists, Instagram feeds, Twitter feeds, countdowns and text messages. All of these different types can be located in an TV or in a group of TVs and be perfectly synchronized.
